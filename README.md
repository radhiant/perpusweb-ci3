# Perpusweb-ci3

Load Mpdf

1. Download Mpdf with composer
2. Composer command "composer require mpdf/mpdf"

Framework used : Codeigniter 3.1.11

CSS Template : SB Admin 2

js: jquery, ajax

plugin js :

- SweatAlert2
- Datepicker
- Datatable
- Chosen

features:

- Dashboard
- Profil
- User

---Data Master---
- Data buku
- Data kategori
- Data Penerbit
- Data Rak
- Data Anggota
- Data Pengadaan Buku

---Transaksi---
- Transaksi Peminjaman
- Transaksi Pengembalian

---Laporan---
- Laporan Peminjaman
- Laporan Pengadaan Buku
